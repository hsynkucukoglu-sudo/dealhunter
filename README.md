# DealhunterMarket

Hollanda market zincirlerinin haftalık indirimli ürünlerini gösteren kişisel dashboard.

## Teknoloji Stack

- **Frontend**: React 19, TypeScript, Vite, Tailwind CSS 4, Framer Motion
- **Backend**: Node.js, Express
- **Database**: SQLite
- **Animation**: Framer Motion

## Kurulum

```bash
npm install
npm run dev
```

Frontend: http://localhost:5173
Backend: http://localhost:3001

## Özellikler

- ✨ Ürün ekleme formu
- 📦 İndirim listesi
- 🎨 Animasyonlar
- 📱 Responsive tasarım
- 🔍 İndirim arama
